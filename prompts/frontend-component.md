# Frontend Prompt

Use this prompt from the `clean-code-lab` root.

```text
Use clean-code-standards.

Create a new frontend project in workspaces/task-dashboard.

Build a small task dashboard with:
- a task list
- status filters
- a form to add a task
- derived counts by status

Keep the project simple. Choose reasonable tooling. The goal is to evaluate code organization, naming, constants, component boundaries, and readable state flow.

Before coding, state the slice boundary. After coding, run available checks and report the Clean Code Gate.
```
