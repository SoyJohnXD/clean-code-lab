# Backend Prompt

Use this prompt from the `clean-code-lab` root.

```text
Use clean-code-standards.

Create a new backend project in workspaces/inventory-api.

Build a small inventory API with:
- create product
- list products
- reserve stock
- reject reservations that exceed available stock

Use in-memory persistence for now. Keep domain rules separate from HTTP/framework code. The goal is to evaluate SOLID, naming, domain boundaries, magic values, and tests.

Before coding, state the slice boundary. After coding, run available checks and report the Clean Code Gate.
```
