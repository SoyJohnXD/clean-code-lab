# Todo App

Small dependency-free Todo app for the Clean Code Lab.

## Run

Open `index.html` in a browser, or serve the folder with any static file server.

## Check

```bash
npm test
```

## Scope

- Add todos.
- Mark todos complete or active.
- Filter all, active, or completed todos.
- Delete a todo.
- Clear completed todos.
- Persist todos in browser `localStorage`.
